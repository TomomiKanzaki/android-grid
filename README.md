# android-grid
### ５つのタブにそれぞれview(タイムテーブル)配置
- 表示されていないviewのメモリ開放を行うことでスムーズなスクロールを可能に
- 縦横同時スクロール対応
- タブの下に配置された時間をタップすることでその時間までテーブルを自動スクロール
